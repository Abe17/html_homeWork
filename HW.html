Latitud vs Temperature

Visualization

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

After assembliing the dataset, we used Matplotlib to plot various aspects of the weather vs latitud. Factors we looked at included: emperature, cloudness, wind speed, and humidity. This site provides the source data and visualization created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
